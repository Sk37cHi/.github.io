Supports iOS 7.0 - 8.4

3 4 All [Glyphs Fuel]

Sk37cH's vector-made glyphs, batch integrated to the great Fuel icon base by DjeTouch. Thank you both very much for allowing us to use them for this set. Sk37cH's carefully planned batch process allows for so much versatility without the need for IconOmatic! An IconOmatic theme is included so you have the option to run DjeTouch's matching original shadow, and enjoy the full Fuel effect. I couldn't resort to Sketch's approach of integrating the shadow to the AppIconMask file due to the image's dimensions not fitting within iOS' restrictions for that file.

Sk37cH's glyphs aren't batch resized thoughout the various devices. This was done intentionally, to ensure a really crisp anti-aliasing throughout the custom icons made to suit the 120px canvas.

Compatible with iPhone4/5/6 running iOS8. For a vector resized and optimised version for iPhone6+, please see "3 4 All [Glyphs Fuel][iP6]" in Cydia.

 

::  Features ::

  • New Flat mod available in Cydia (v6).
  • All Stock icons themed (including iOS8.4).
  • Settings and NC icons, including iOS8x.
  • Over 250 apps themed including third party icons, with more to come.
  • A unique execution of theming on iOS8 (seprate iP6 Plus 180px icon output).
  • Classic badge included.
  • McFly 5 Day forecast iWidgets available on the MMi forum.
